# Blockchain Web Application Using Web3 JS [Platform - ETH]

A basic blockchain application made using web3js, node, angular, mongodb.

## Features

1. Ability to Create Account
2. Store Account Details in DB
3. Check Balance
4. Make Transaction

## Instructions to Run

**Frontend**

1. `cd fronted`
2. `npm i`
3. `ng s`

**Backend**

1. `cd backend`
2. `npm i`
3. `cd src`
4. `nodemon`
